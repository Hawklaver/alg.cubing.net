AnimCube + Heise Sim: AlgTyper
by Lucas Garron, November 29, 2008; Initial Test Version



Thanks to Ryan Heise for creating a simulator with a good keyboard control of the cube, and to Josef Jelinek for writing a well-commented applet.
Note that this is a complete hack. It's only designed for obtaining the notation of an alg from muscle memory. Fast typing will sometimes drop moves. Verify any alg before using.
Directions:
Select the applet, and perform moves on it like in the Heise sim.
Press "Gimme Alg!" The text should appear in the input field -selected, ready to copy.
